Management Information System - ICT2132 Mini Project

University of Ruhuna - Faculty of Technology

Project Description

This project is a Java and MySQL-based application developed to manage academic and administrative tasks for the Faculty of Technology.

Features

  • Admin: Manage user profiles, course details, notices, and timetables.
  • Lecturer: Update profile, manage course materials, upload marks, view grades and attendance.
  • Technical Officer: Update profile, maintain student attendance and medical records.
  • Student: Update contact details, view attendance, grades, timetable, and notices.
